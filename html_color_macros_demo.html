HTML Colors Demonstration

This table is built with the specification of three colors: Panel 1 Panel 2 Panel 3. We use color math macros to blend these together for the "overlapping" bits.

                             
                             
                             
                   Wow, remember when HTML tables were used for layout?       
                             

Of course, this demo is showing this project's age. In the modern, civilized world, one might use color math in their CSS templates rather than build colored tables. Which one could totally do with m4-bakery, by creating files named something.css.m4.
